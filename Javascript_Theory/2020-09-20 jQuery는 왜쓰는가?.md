### jQuery 란?

```
* jQuery 란, javascript 로 만든 라이브러리가 jQuery 입니다.

* 사용법 
  <script src="https://code.jquery.com/jquery-3.4.1.js"></script>

* 장점
  1. 대부분의 브라우저와 버전을 지원합니다. (멀티 브라우저 지원)
  2. 코드의 길이가 굉장히 짧아집니다.
  
* 단점
  1. jQuery 뿐만 아니라, 각 라이브러리마다 고유의 작성법이 있으므로, 문법을 이해할 수 없다면 라이브러리를 사용못한다.
  2. 라이브러리는 다양한 기능이나, 효과, 연출을 구현하는 Javascript 코드 모음이므로, 웹 페이지를 열때 해당 코드를 모두 읽어야 하기 때문에 
     웹 페이지의 로딩 시간이 길어진다는 단점이 있다.

```
