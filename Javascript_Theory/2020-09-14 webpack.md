```
webpack 이란?

- 현재 가장 널리 쓰이는 번들러이며, 자바스크립트뿐만 아니라 CSS, Image 파일 등 리소스의 의존성도 관리한다.

+ 예를 들어, CSS/Sass/Less 내부에서 사용하는 @import , url(...) 구문이나 HTML 내부의 <img src=...> 태그를 모두 관리한다.

```
